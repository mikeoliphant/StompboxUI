This is a VST3 plugin, and as such must be run within a VST host such as
a DAW (Digital Audio Workstation) application.

For your host to see the plugin, the "Stompbox" folder must be copied to your
VST3 path - usually "C:\Program Files\Common Files\VST3".

**Important Note**: You must have .NET 8.0 or later installed:

https://dotnet.microsoft.com/en-us/download/dotnet/8.0
